# cellularism
cellularism is a project built in the processing ide which simulates particles in a 3d environment that interact with eachother creating emergent behavior.
## how do i install/build the project?
1. first [install the processing IDE](https://processing.org/download)
2. locate your sketchbook directory, for linux this is `home/user/sketchbook`
3. download the project zip file and extract in the directory from step 2
4. open sketch in processing
5. run the program!
